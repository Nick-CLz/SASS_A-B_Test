Common questions

Can I trust an LLM with statistics?

The LLM never computes a number. The engine computes; the agent narrates; every figure is traced to a tool call (grounding) and recorded. It's auditable, and the math is deterministic.

Our data can't leave our environment.

Analytics run warehouse-native, so raw data stays in your perimeter. Enterprise adds full self-host / VPC deployment with a DPA and security review.

We already use feature flags (LaunchDarkly / Split).

Keep them. Mallard is the analysis-and-brain layer, not a flag tool — we complement flagging with rigorous analysis and an agentic workflow.

How is this different from Statsig or Eppo?

Same warehouse-native rigor, plus AI agents that actually do the data-scientist workflow and a privacy-first posture built in from the data model up.

How do you keep AI costs sane?

Model routing (small/medium/large per task) and prompt caching. AI usage is metered to customers, so cost scales with value delivered.
